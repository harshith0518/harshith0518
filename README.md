## Featured Work

### Online Judge System

An end-to-end competitive programming platform built with Django, React.js, Docker, and AWS.

- Designed REST APIs for problem management, code compilation, submissions, evaluation, and leaderboards.
- Implemented JWT-based authentication and full CRUD flows.
- Used Docker and Docker Compose for local development and deployment flow.
- Worked on the direction for a custom compiler backend to reduce reliance on third-party execution APIs.
- Built with HLD and LLD thinking, with mentorship from engineers across Apple, Google, Grab Singapore, and AlphaGrep Singapore.

### Data Analysis of a Complex System

A research-style simulation project under Prof. Danny Raj using Python, PySINDy, NumPy, Matplotlib, and research papers.

- Simulated an SIR model in an urban-like environment generated through a core-periphery layout heuristic.
- Produced video outputs to study spread dynamics visually.
- Designed a metric for structural emergence and alignment.
- Explored equation discovery with SINDy on synthetic simulation data.

### Sports Players Face Recognition

An ML-based face classification project using OpenCV, Scikit-learn, NumPy, and Matplotlib.

- Built a classifier for sports-player face recognition across cricket, tennis, football, and badminton.
- Preprocessed 200+ images with grayscale conversion and histogram equalization.
- Used PCA for dimensionality reduction and compared multiple classifiers for performance.
